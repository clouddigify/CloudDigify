# CloudDigify

A modern responsive homepage for CloudDigify, built with React, Vite, Tailwind CSS, and Framer Motion.

## Getting Started

### Prerequisites
- Node.js (>=14)

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
Open http://localhost:5173 in your browser.

### Production Build
```bash
npm run build
```

### Preview Production
```bash
npm run preview
```
